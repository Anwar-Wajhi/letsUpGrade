Q2. Difference between var, let and const with examples

Ans.

var - Var is a keyword used to declare variables.
Variables declared with the var keyword cannot have Block Scope.
Variables declared inside a block {} can be accessed from outside the block. 

                function getDate () {
                var date = new Date()

                return date
                }

                getDate()
                console.log(date)

let - Let is a keyword used to declare variables.
Variables declared with the let keyword can have Block Scope.
Variables declared inside a block {} cannot be accessed from outside the block: 

                {
                    var x = 2;
                  }
                  // x can be used here

Const - Declaring a variable with const is similar to let when it comes to Block Scope except they cannot be reassigned.
The x declared in the block, in this example, is not the same as the x declared outside the block: 

                const PI = 3.141592653589793;
                PI = 3.14;      // This will give an error
                PI = PI + 10;   // This will also give an error



Q3. Write a brief intro on available DataTypes in JavaScript

Ans.

JavaScript includes primitive and non-primitive data types as per ECMAScript.


Primitive Data Types

  • String
  • Number
  • Boolean
  • Null
  • Undefined
Non-primitive Data Types
  • Object
  • Date
  • Array
JavaScript is a dynamic or loosely-typed language because a variable can hold value of any data type at any point of time.
